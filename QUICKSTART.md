# ⚡ Quick Start - Deploy in 10 Minutes

Follow these exact steps to deploy your app:

## 1️⃣ Commit Your Code (2 minutes)

```bash
git add .
git commit -m "Initial commit: Blood Donation Management System"
```

## 2️⃣ Create GitHub Repo (2 minutes)

1. Go to: https://github.com/new
2. Name: `blood-donation-system`
3. Make it **Public**
4. Click "Create repository"
5. Copy the commands shown and run them

## 3️⃣ Deploy to Vercel (3 minutes)

1. Go to: https://vercel.com/signup
2. Sign in with GitHub
3. Click "Add New" → "Project"
4. Select `blood-donation-system`
5. Add environment variables:
   - `SUPABASE_URL`: (from your .env file)
   - `SUPABASE_KEY`: (from your .env file)
6. Click "Deploy"

## 4️⃣ Update Supabase (2 minutes)

1. Go to your Supabase dashboard
2. Authentication → URL Configuration
3. Add your Vercel URL (shown after deployment)

## 5️⃣ Test It! (1 minute)

Visit your live URL and try logging in!
